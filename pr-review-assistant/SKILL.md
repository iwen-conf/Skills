# pr-review-assistant

## Overview

将 PR 评审从“个人风格”升级为“标准化维度”。

## When to Use

- 合并前质量把关
- 团队评审标准不一致
- 需要高质量、可追踪的评审意见

## Outputs

- 评审维度检查表
- 关键风险与阻塞项
- 测试证据充分性结论
- 可执行修改建议（按优先级）

## Workflow

1. 按正确性/可维护性/风险/测试证据四维审查
2. 区分阻塞项与建议项
3. 给出可执行修改建议与优先级
4. 输出评审结论与合并建议
