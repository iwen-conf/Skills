# security-fastcheck

## Overview

以低成本快速发现高风险安全问题并给出优先修复路径。

## When to Use

- 发布前安全快检
- 新仓库接入安全基线
- 安全事件后做范围排查

## Outputs

- 高危项清单（优先级排序）
- 漏洞与误配置证据
- 修复建议与时效要求
- 可接受风险与例外说明

## Workflow

1. 扫描依赖漏洞与高危配置
2. 检查密钥泄漏与权限暴露风险
3. 评估影响范围与修复优先级
4. 输出基线报告与处置建议
